# ChatBotAI

Un site web simple présentant un assistant de chat intelligent avec un design moderne.

## Description

Ce projet contient une page d'accueil, une page "À propos" et une page de contact, toutes intégrant une vidéo de fond et une navigation claire.

## Pages principales

- `home.html` : page d'accueil avec vidéo en arrière-plan, titre, logo et bouton d'accès au chat.
- `about.html` : page de présentation de la mission, de la vision et des avantages de ChatBotAI.
- `contact.html` : page de contact avec informations de support et lien vers le chat.
- `index.html` : page de démarrage du chat (généralement le point d'entrée vers l'application de conversation).

## Styles

- `home.css` : styles partagés pour l'ensemble du site, y compris la vidéo de fond, la navigation, les boutons et les sections de contenu.
- `index.css` : styles supplémentaires pour la page de chat.

## Ressources

- `assets/video/backround.mp4` : vidéo de fond utilisée par les pages.
- `assets/images/favicon.png` : logo et image de marque.

## Utilisation

1. Ouvrir `home.html` dans un navigateur.
2. Naviguer vers `about.html` ou `contact.html` depuis la barre de navigation.
3. Cliquer sur le bouton pour accéder à `index.html` et lancer le chat.

## Notes

- Le site est conçu pour être statique et facile à modifier.
- Les pages utilisent une vidéo en arrière-plan et des éléments superposés pour un rendu visuel attrayant.
